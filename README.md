# Movie Recommender
## Motivation
To make a basic app to recommed movies
## Requirements
Python 3.8 or above with all [requirements](requirements.txt) dependencies installed. To install run:
```python
$ pip3 install -r requirements.txt
```
## To run
```python
$ streamlit run movie.py
```
